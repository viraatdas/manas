what it does

Todos judged against real work

Each todo gets a verdict — done, in progress, not started — with one line of evidence pulled from your local Claude Code and Codex transcripts.

Discovered activities

Feature-level work found in your sessions that never made the list. Add it, already checked off, or dismiss it for good.

A day journal with a price tag

Automatic check-ins on launch and every hour. A usage strip shows today's tokens, cost, and checks, with a per-check table and 7-day sparkline behind it.

Calendar aware

Your Mac calendar in the flow of the day — and typing “@6:30pm” on a todo puts a real event on it.

Where the time went

Arc browsing history and Screen Time fill in the hours between commits, so the day adds up honestly.

Message awareness

Knows when the afternoon disappeared into iMessage, and says so without judgment. Well — with exactly one line of judgment.

local first

No API keys, no accounts, no telemetry. Manas reads local files — ~/.claude/projects and ~/.codex/sessions — and judging shells out to the claude CLI you already have, on your existing subscription.

Built with SwiftUI, Swift Charts, and SF Symbols. Requires macOS 14+. Manas is in active development.
